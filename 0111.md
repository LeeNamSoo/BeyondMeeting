
# Git
### Git 저장소 주소 repository 변경하기
``` bash
git remote set-url origin <새로운git url>

``` 
### 브랜치 생성
``` bash
git branch <브랜치이름>
```

### 브랜치로 이동
``` bash
git checkout <브랜치이름>
```

### 브랜치에서 push
``` bash
git push origin <브랜치이름>
```
